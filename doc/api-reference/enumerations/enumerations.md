## Description

Describes all available enumerations.

## List of Enumerations

> [DsVeosCoSim_CanMessageFlags](DsVeosCoSim_CanMessageFlags.md)

Contains the possible flags of a CAN message.

> [DsVeosCoSim_Command](DsVeosCoSim_Command.md)

Contains commands for controlling the co-simulation in polling mode.

> [DsVeosCoSim_ConnectionState](DsVeosCoSim_ConnectionState.md)

Contains the possible connection states.

> [DsVeosCoSim_DataType](DsVeosCoSim_DataType.md)

Contains valid data types for I/O signals.

> [DsVeosCoSim_EthMessageFlags](DsVeosCoSim_EthMessageFlags.md)

Contains the possible flags of an Ethernet message.

> [DsVeosCoSim_FrMessageFlags](DsVeosCoSim_FrMessageFlags.md)

Contains the possible flags of a FlexRay message.

> [DsVeosCoSim_LinControllerType](DsVeosCoSim_LinControllerType.md)

Contains the LIN controller type.

> [DsVeosCoSim_LinMessageFlags](DsVeosCoSim_LinMessageFlags.md)

Contains the possible flags of a LIN message.

> [DsVeosCoSim_Result](DsVeosCoSim_Result.md)

Contains the possible return values of VEOS CoSim functions.

> [DsVeosCoSim_Severity](DsVeosCoSim_Severity.md)

Contains all possible severity levels of a log message in descending order of severity.

> [DsVeosCoSim_SimulationState](DsVeosCoSim_SimulationState.md)

Contains the possible simulation states.

> [DsVeosCoSim_SizeKind](DsVeosCoSim_SizeKind.md)

Contains information on the I/O signal length type.

> [DsVeosCoSim_TerminateReason](DsVeosCoSim_TerminateReason.md)

Contains the possible reasons for terminating the co-simulation.
